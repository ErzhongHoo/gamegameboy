# 🎮 GameBoy 项目总结

## 📝 使用指南

### 快速开始
1. 编辑 `scripts/generate-pages.js` 添加真实游戏
2. 运行 `node scripts/generate-pages.js` 生成页面
3. 部署到任意静态托管平台

### 自定义配置
- 修改颜色主题：编辑 `assets/css/style.css`
- 调整游戏分类：修改生成脚本
- 更新网站信息：编辑各HTML文件的meta标签
